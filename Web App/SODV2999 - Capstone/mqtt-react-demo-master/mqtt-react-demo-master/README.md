## Documentation

```
mqtt sub -t 'camplug/1' -h a3ieilx72a5bln-ats.iot.us-west-1.amazonaws.com
```

Publish in Terminal
```
mqtt pub -t 'camplug/1' -h a3ieilx72a5bln-ats.iot.us-west-1.amazonaws.com -m 'from commandline'
```
